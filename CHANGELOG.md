# Changelog

All notable changes to the QPerfect documentation portal are recorded here.
The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and the project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-14

### Added

- QPerfect documentation hub at the root, with a Products grid built to
  extend beyond MIMIQ.
- MIMIQ sub-hub at `/mimiq/` linking the MIMIQ Python, MIMIQ Qiskit
  Integration, and MIMIQ Julia interfaces.
- Shared `assets/portal.css` styled to the qperfect.io brand tokens.
- Qiskit interface icon in the QPerfect palette.

### Changed

- Rebuilt the landing page from a single API picker into the documentation
  hub, on the teal and amber brand palette with Plus Jakarta Sans and Inter.
- Footer Contact now opens the qperfect.io contact form; external links open
  in a new tab.
